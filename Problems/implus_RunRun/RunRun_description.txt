Description:
王小可在另外一个操场参加跑步比赛。所有童鞋在四周热身。比赛终点也是一条直线，万万妹想到，终点线的向量v是固定的，但是经过哪一个点是由裁判随机决定的。王小可给每人吃了一个苹果，所有人都答应他跑步的时候和他保持一样的速度。王小可天生匀速，他跑步的时候速度正好是单位1。问，裁判确定终点经过点之后一声令下，所有同学依次从热身点出发，以最快地方式跑到终点线的总时间是多少？（依次跑就是等一个人跑完，另外一个人再开始跑）

Input：
Multiple cases，end with EOF.
第一行，x y， 终点线哒非零方向向量。
下面一行一个正整数n （1 <= n <= 10^5), 表示n个童鞋。
接下来n行，每行两个整数，x y，表示n个童鞋的坐标。
下面一行一个正整数m （1 <= m <= 10^5), 表示裁判的m次可能的命令。
接下来m行，每行两个整数，x y，表示m种不同的终点可能经过的点。
(所有数字的绝对值不会超过100000）

Output：
对于每一次裁判可能的命令，输出要求的总时间,保留6位小数。

Sample Input：
0 1
2
1 0
2 1
3
0 0
1 0
4 0

Sample Output:
3.000000
1.000000
5.000000
